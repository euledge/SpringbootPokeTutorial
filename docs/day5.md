## 4. 作業手順の詳細

### Step 1: カードデザインの改善
1. `Card.vue` のCSSを修正し、ポケモンのタイプ（Fire, Waterなど）に応じて背景色が変わるようにします。
   - クラスバインディング `:class="card.type"` を活用します。
2. ホバー時に少し浮き上がるようなエフェクトを追加します。
   ```css
   .card:hover {
     transform: translateY(-5px);
     box-shadow: 0 10px 20px rgba(0,0,0,0.2);
   }
   ```

### Step 2: 開封アニメーション
1. パック画像を用意し、クリックすると揺れてからカードが飛び出すような演出を作ります。
2. Vueの `<Transition>` コンポーネントや、CSSの `@keyframes` を使って実装します。
   - 例: カードが表示される瞬間に `fade-in` クラスを適用する。

### Step 3: ローディングとエラー処理
1. `PackView.vue` に `isLoading` と `error` というリアクティブ変数を追加します。
2. API呼び出しの前後で `isLoading` を `true/false` に切り替えます。
3. `try-catch` ブロックでエラーを捕捉し、エラー時はメッセージを表示します。
   ```html
   <div v-if="isLoading">Opening pack...</div>
   <div v-else-if="error" class="error">{{ error }}</div>
   ```

### Step 4: 最終確認
1. アプリ全体を通して操作し、バグがないか確認します。
2. コードを見直し、不要なログ出力やコメントを削除します。
3. `README.md` に完成した機能やスクリーンショットを追記しても良いでしょう。
