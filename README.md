# PokeTutorial

SpringBootとVue.jsで学ぶ、ポケモンカード収集アプリ「PokeCa Pocket Lite」チュートリアル。

## プロジェクト概要
このリポジトリは、Spring Boot (Backend) と Vue.js (Frontend) の連携を学ぶためのハンズオン用コードベースです。
PokeAPIを利用して、パック開封シミュレーションとカード収集機能を実装します。
